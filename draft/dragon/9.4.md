### 9.4.4 単調性 $x < y → f(x) < f(y)$
ある変数のみに着目して，それが単調なら良い．

### 9.4.5 非分配性 (Nondistributivity)
distributive : $f(a ^ b) = f(a) ^ f(b)$

a : { x -> 2, y -> 3}
b : { x -> 3, y -> 2}
a ^ b : { x -> NOC, y -> NOC }
f(a ^ b) : {x -> NOC, y -> NOC, z -> NOC }
f(a) : {x -> 2, y -> 3, z -> 5}
f(b) : {x -> 3, y -> 2, z -> 5}
f(a) ^ f(b) : { x -> NOC, y -> NOC, z -> 5 }
